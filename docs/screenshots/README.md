Automatisk generering via Playwright

- Kør: `pnpm test:screens`
- Krav: app kører lokalt på `http://localhost:3000` og dev-login er tilgængelig på `/api/auth/login`.
- Output gemmes i denne mappe under de viste sti‑navne.
