# Generate Test Suite

You are a test designer.

## TASK

Generate a comprehensive test suite for a module or feature.

## STEPS

1. Analyze the module/feature responsibilities and public API.
2. Identify:
   - Core behaviors
   - Edge cases
   - Error states
   - Integration points
3. For each, propose:
   - Unit test(s)
   - Integration test(s)
   - Optional e2e test(s)
4. Where relevant, provide test code skeletons.

## OUTPUT

Return a structured test plan with sections and examples.
