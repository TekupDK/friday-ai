# Test Samtale

Test denne samtale: verificer at commands virker korrekt, test chat flow, og valider at alt fungerer som forventet.

## ROLE & CONTEXT

- **Project:** Friday AI Chat (TekupDK/Rendetalje.dk)
- **Stack:** React 19 + TypeScript + tRPC 11 + Drizzle ORM
- **Purpose:** Test denne samtale og commands i praksis
- **Quality:** Praktisk, valideret, actionable

## TASK

Test denne samtale:
1. **Læs chat historik** - HELE samtalen fra start
2. **Test commands** - Kør commands og verificer de virker
3. **Valider output** - Tjek at output er korrekt
4. **Test edge cases** - Test med forskellige inputs
5. **Rapporter resultater** - Dokumenter hvad der virker og ikke virker

## CHAT SESSION READING

**KRITISK:** Læs HELE samtalen:
- Start fra første besked
- Læs ALLE beskeder fra brugeren
- Læs ALLE svar fra agenten
- Forstå hvad der er testet
- Identificér hvad der virker og ikke virker

## IMPLEMENTATION STEPS

1. **Læs Chat Historik:**
   - Start fra første besked i denne samtale
   - Læs ALLE beskeder i rækkefølge
   - Forstå hvad der er diskuteret
   - Identificér test cases

2. **Test Commands:**
   - Kør hver command der er nævnt
   - Verificer at de virker korrekt
   - Tjek output kvalitet
   - Identificér problemer

3. **Valider Output:**
   - Tjek at output matcher forventninger
   - Verificer at commands følger guidelines
   - Tjek at output er actionable
   - Valider at alt er korrekt

4. **Test Edge Cases:**
   - Test med forskellige inputs
   - Test med manglende information
   - Test med ugyldige inputs
   - Test med komplekse scenarier

5. **Rapporter Resultater:**
   - Dokumenter hvad der virker
   - Dokumenter hvad der ikke virker
   - Giv konkrete forbedringer
   - Prioriter fixes
