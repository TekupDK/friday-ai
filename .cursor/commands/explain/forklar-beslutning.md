# Forklar Beslutning

Forklar en beslutning: kontekst, rationale, alternativer, og impact.

## ROLE & CONTEXT

- **Project:** Friday AI Chat (TekupDK/Rendetalje.dk)
- **Stack:** React 19 + TypeScript + tRPC 11 + Drizzle ORM
- **Purpose:** Forklar beslutninger på klar, struktureret måde
- **Quality:** Klar, nøjagtig, forståelig

## TASK

Forklar beslutning:
1. **Kontekst** - Hvad var situationen?
2. **Rationale** - Hvorfor blev denne beslutning taget?
3. **Alternativer** - Hvad kunne være gjort anderledes?
4. **Impact** - Hvad betyder det?

## OUTPUT FORMAT

```markdown
## Beslutning

[Beskrivelse af beslutningen]

## Kontekst

[Situationen der førte til beslutningen]

## Rationale

- [Grund 1]
- [Grund 2]

## Alternativer

- [Alternativ 1] - [Hvorfor ikke valgt]
- [Alternativ 2] - [Hvorfor ikke valgt]

## Impact

- [Impact 1]
- [Impact 2]
```

## GUIDELINES

- **Klar:** Forklar i simple termer
- **Nøjagtig:** Fokusér på faktisk beslutning
- **Balanceret:** Nævn både fordele og ulemper
- **Kortfattet:** Maks 25 linjer

---

**CRITICAL:** Forstå beslutningen, identificér kontekst og rationale, og forklar på klar, struktureret måde.
