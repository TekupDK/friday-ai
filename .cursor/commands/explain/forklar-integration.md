# Forklar Integration

Forklar en integration: hvordan den virker, endpoints, data flow, og hvorfor den er vigtig.

## ROLE & CONTEXT

- **Project:** Friday AI Chat (TekupDK/Rendetalje.dk)
- **Stack:** React 19 + TypeScript + tRPC 11 + Drizzle ORM
- **Purpose:** Forklar integrationer på klar, struktureret måde
- **Quality:** Klar, komplet, forståelig

## TASK

Forklar integration:
1. **Hvordan virker den?** - Flow og endpoints
2. **Data flow** - Hvordan flyder data?
3. **Hvorfor er den vigtig?** - Business value

## OUTPUT FORMAT

```markdown
## Integration

[Kort beskrivelse af integrationen]

## Hvordan virker den?

1. [Step 1]
2. [Step 2]
3. [Step 3]

## Endpoints

- `[method] [path]` - [Beskrivelse]
- `[method] [path]` - [Beskrivelse]

## Data Flow

[Beskrivelse af data flow]

## Business Value

- [Value 1]
- [Value 2]
```

## GUIDELINES

- **Klar:** Forklar i simple termer
- **Komplet:** Dæk alle endpoints og flow
- **Forståelig:** Ingen jargon
- **Kortfattet:** Maks 30 linjer

---

**CRITICAL:** Forstå integrationen, identificér endpoints og data flow, og forklar på klar, struktureret måde.
