# Forklar Fejl

Forklar en fejl: hvad er problemet, hvorfor opstår det, og hvordan fixes det.

## ROLE & CONTEXT

- **Project:** Friday AI Chat (TekupDK/Rendetalje.dk)
- **Stack:** React 19 + TypeScript + tRPC 11 + Drizzle ORM
- **Purpose:** Forklar fejl på klar, forståelig måde
- **Quality:** Klar, nøjagtig, actionable

## TASK

Forklar fejl:
1. **Hvad er problemet?** - Fejlbeskrivelse
2. **Hvorfor opstår det?** - Root cause
3. **Hvordan fixes det?** - Løsning

## OUTPUT FORMAT

```markdown
## Problem

[Kort beskrivelse af problemet]

## Root Cause

[Forklaring af hvorfor fejlen opstår]

## Løsning

1. [Step 1]
2. [Step 2]
3. [Step 3]

## Prevention

- [Prevention 1]
- [Prevention 2]
```

## GUIDELINES

- **Klar:** Forklar i simple termer
- **Nøjagtig:** Fokusér på faktisk fejl
- **Actionable:** Giv konkrete løsninger
- **Kortfattet:** Maks 20 linjer

---

**CRITICAL:** Forstå fejlen, identificér root cause, og forklar løsning på klar, actionable måde.
