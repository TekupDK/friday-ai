# Track Todo Progress

You are tracking and reporting progress on TODO items.

## TASK

Review TODO list and provide a progress report on completion status.

## STEPS

1) Read the TODO list:
   - Parse all tasks
   - Note status markers
   - Check completion indicators

2) Categorize tasks:
   - Completed: `- [x]` or `✅`
   - In Progress: `- [~]` or `⏳`
   - Not Started: `- [ ]`
   - Blocked: `- [!]` or `⚠️`

3) Calculate metrics:
   - Total tasks
   - Completed count
   - In progress count
   - Blocked count
   - Completion percentage

4) Analyze by area:
   - Backend progress
   - Frontend progress
   - Tests progress
   - Docs progress
   - Other areas

5) Identify trends:
   - Tasks completed recently
   - Tasks stuck/blocked
   - Areas needing attention
   - Velocity indicators

6) Provide recommendations:
   - Focus areas
   - Unblocking actions
   - Next priorities
   - Risk items

## OUTPUT

Provide:
- Progress summary with metrics
- Breakdown by area
- Status of each task
- Trends and insights
- Recommendations
- Updated TODO list if statuses changed
