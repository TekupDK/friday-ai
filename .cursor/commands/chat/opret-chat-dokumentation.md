# Opret Chat Dokumentation

Lav en pæn `.md` med recap, plan, tasks og beslutninger.

## ROLE & CONTEXT

- **Project:** Friday AI Chat (TekupDK/Rendetalje.dk)
- **Stack:** React 19 + TypeScript + tRPC 11 + Drizzle ORM
- **Purpose:** Opret struktureret dokumentation fra chatten
- **Quality:** Komplet, struktureret, klar

## TASK

Lav en pæn `.md` fil med:

- Recap: Hvad blev diskuteret?
- Plan: Hvad er planen?
- Tasks: Hvad skal gøres?
- Beslutninger: Hvad blev besluttet?

## OUTPUT FORMAT

```markdown
# [Titel]

## Recap

[Kort beskrivelse af hvad der blev diskuteret]

## Plan

1. [Step 1]
2. [Step 2]
3. [Step 3]

## Tasks

- [ ] [Task 1]
- [ ] [Task 2]

## Beslutninger

- [Beslutning 1] - [Rationale]
- [Beslutning 2] - [Rationale]
```

## GUIDELINES

- **Struktureret:** Klar struktur med sektioner
- **Komplet:** Dæk recap, plan, tasks, beslutninger
- **Klar:** Let at forstå
- **Actionable:** Tasks skal være konkrete

---

**CRITICAL:** Læs chatten, ekstraher recap, plan, tasks og beslutninger, og opret en pæn `.md` fil.
