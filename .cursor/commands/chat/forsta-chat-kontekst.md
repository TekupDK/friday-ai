# Forstå Chat Kontekst

Giv en dybere forklaring: hvad betyder alt dette for arkitektur og flow?

## ROLE & CONTEXT

- **Project:** Friday AI Chat (TekupDK/Rendetalje.dk)
- **Stack:** React 19 + TypeScript + tRPC 11 + Drizzle ORM
- **Purpose:** Dyb forståelse af arkitektur og flow impact
- **Quality:** Insightful, dyb, klar

## TASK

Giv dyb forklaring:

- Hvad betyder dette for arkitektur?
- Hvad er flow impact?
- Hvad er dependencies?
- Hvad er risici?

## OUTPUT FORMAT

```markdown
## Arkitektur Impact

[Kort beskrivelse af hvad dette betyder for arkitektur]

## Flow Impact

[Beskrivelse af hvordan dette påvirker flow]

## Dependencies

- [Dependency 1] - [Impact]
- [Dependency 2] - [Impact]

## Risici

- [Risk 1] - [Mitigation]
- [Risk 2] - [Mitigation]
```

## GUIDELINES

- **Dyb:** Gå dybt i forståelse
- **Arkitektur-fokuseret:** Fokusér på arkitektur impact
- **Klar:** Forklar i simple termer
- **Kortfattet:** Maks 25 linjer

---

**CRITICAL:** Læs chatten, forstå dybt hvad dette betyder for arkitektur og flow, og giv klar forklaring.
