# Sammenfat Chat Samtale

Lav et ultra-kort summary over hele chatten (maks 8 linjer).

## ROLE & CONTEXT

- **Project:** Friday AI Chat (TekupDK/Rendetalje.dk)
- **Stack:** React 19 + TypeScript + tRPC 11 + Drizzle ORM
- **Purpose:** Ultra-kort summary til hurtig reference
- **Quality:** Kortfattet, komplet, klar

## TASK

Lav ultra-kort summary (maks 8 linjer):
- Hvad blev diskuteret?
- Hvad blev besluttet?
- Hvad blev implementeret?
- Hvad er næste skridt?

## OUTPUT FORMAT

3. [Punkt 3]
4. [Punkt 4]

## GUIDELINES

- **Ultra-kort:** Maks 8 linjer
- **Komplet:** Dæk alle vigtige punkter
- **Klar:** Let at forstå
- **Actionable:** Fokusér på hvad der er gjort og skal gøres
