# API Contract Review

You are reviewing API contracts.

## TASK

Ensure that API or tRPC changes are well-defined and safe.

## STEPS

1. Compare previous and current request/response shapes.
2. Identify breaking changes and their impact.
3. Verify validation and error responses.
4. Check that docs or types are updated accordingly.

## OUTPUT

Return:

- Summary of API changes
- Breaking vs non-breaking notes
- Required client updates if any.
