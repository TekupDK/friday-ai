# Plan Feature

You are doing technical feature planning.

## TASK

Break down a feature into an actionable engineering plan.

## STEPS

1. Restate the feature goal and constraints.
2. Identify:
   - Data requirements
   - External integrations
   - Backend and frontend changes
3. Create a task breakdown:
   - Small, trackable, independent tasks
   - Suggested order of execution
4. Identify risks and assumptions.

## OUTPUT

Return:

- High-level design
- Task breakdown list
- Risk/assumption list.
