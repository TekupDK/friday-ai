# Split Large File

You are refactoring a large file into smaller, focused modules.

## TASK

Split a large file into logically coherent parts.

## STEPS

1. Identify logical groupings (domains, features, utility groups).
2. Create new files for each grouping.
3. Move code while preserving public interfaces.
4. Update imports.
5. Run tests and typechecks.

## OUTPUT

Provide:

- New file structure
- Rationale for the split
- Any potential migration concerns.
