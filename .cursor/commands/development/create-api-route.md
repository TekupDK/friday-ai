# Create API Route

You are creating a new API route (REST or tRPC).

## TASK

Design and implement a new route for a specific use case.

## STEPS

1) Define the route's responsibility in one sentence.
2) Specify:
   - HTTP method / tRPC procedure
   - Input schema (validation)
   - Output schema
   - Auth requirements
3) Implement the handler:
   - Use existing patterns and abstractions
   - Add logging where useful
4) Add tests for:
   - Happy path
   - Invalid input
   - Unauthorized access.

## OUTPUT

Provide:
- Handler/route implementation
- Schema definitions
- Test cases.
